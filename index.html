Hi,I'm Philix!

I'm a UX/UI Designer with a passion for visual arts. I specialize in wireframing, styling, and iterating clickable prototypes of websites and apps.

My UX Skills

User Research

I enjoy working with a team to brainstorm ideas, gather insight from user interviews, and flesh out empathy-driven solutions through problem analysis, persona building, storyboarding, and usability testing of our preliminary wireframes prototypes. Some tools I find useful during this process are Miro, Sketch, Figma, and Maze.

User Interface Design

Transforming a thouroughly researched ideation into something tangible is like breathing life into a collective consiousness. I work with tools such as Figma, Invision and XD to combine the aesthetics of color theory and a balanced layout with the organization of navigation systems and the puzzle-like process of stringing together interactive prototypes.

Web Development

I'm relatively new to the world of coding, but I find it fascinating and am eager to learn more. I am confident in my ability to be code conversational and am working on improving my own front-end development skills. So far, I've practiced using HTML, CSS, JavaScript, jQuery, and Bootstrap. The tools I've worked with include codepen, Visual Studio Code and GitHub.
